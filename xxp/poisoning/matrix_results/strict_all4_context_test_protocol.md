# Strict All-4 Trigger Test Protocol

## Standard

For BadDreamer ASR evaluation, a triggered test sample must satisfy all of the following:

1. The input/context window has exactly 4 frames.
2. All 4 input frames are edited trigger frames containing the yellow-helmet delivery rider.
3. The 4 input frames are consecutive frames from the same nuScenes front-camera scene, so the delivery rider appears as a temporal progression.
4. The 4 target future frames are the original clean no-trigger ground-truth frames.
5. ASR is computed only on these strict all-4-trigger windows; windows with only 1, 2, or 3 triggered context frames are excluded from the ASR denominator.

This is different from the earlier scene-level attack validation split, where every poisoned window from a held-out poisoned scene was evaluated as long as it had at least one triggered context frame.
